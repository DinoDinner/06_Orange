Orange Data Mining Basic 구조

1. Data File을 넣어서 Data를 입력함
-> Feature와 Target(Label)을 설정할 수 있음
-> Data Table/Scatter Plot에서 Raw Data를 시각적으로 확인 할 수 있음.

2. Module에 Data를 연결해서 알고리즘을 통한 Data 분석을 할 수 있음

3. Module을 통해 분석된 Data(Train Set)를 다른 비교 Data(TestSet)와 비교해서 분석의 유효성을 판단할 수 있음.

4. Prediction에서 예측치를 확인 할 수 있음.
-> Visualizer Scatter Plot 에 연결하여 예측 데이터를 결과로 볼 수 있음.

-------------------------------------------------------------------------------

두 알고리즘 모델을 TrainSet에 연결하고, Prediction에 결과를 연결히면,
어떤 알고리즘이 Data에 맞는 모델인지 판단할 수 있다.

Scatter Plot을 통해서 어느 모델이 더욱 Linear에 가까운지 판단.
